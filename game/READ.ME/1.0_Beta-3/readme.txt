2. How to Install/Play
---ZIPファイル版---
1. ZIPファイルを好きなところへ。
2. ZIPファイルを展開してください。
3. 展開したディレクトリの中の "game.exe" が実行ファイルですので、それを実行すれば、ゲームがプレイできます。
---インストーラー版---
1. インストーラーを起動します。
2. インストーラーの指示に従ってください。
3. (ショートカットを作成した場合)スタートメニューやデスクトップのアイコンを実行してください。
    もしくは、インストールしたディレクトリのgame.exeを実行してください。

3. How to Uninstall
---ZIPファイル版---
1. How to Install/Playで展開したディレクトリを丸ごと削除してください。コンパネや設定に行く必要はありません。
---インストーラー版---
1. コントロールパネルを開きます。
2. プログラムをクリックします。
3. プログラムのアンインストールをクリックします。
4. 図書室へ行こう! を探す。
5. メニューバーの アンインストール をクリック。
6. アンインストーラーの指示に従ってください。
